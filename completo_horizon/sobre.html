O que é a Horizon - Orçamentos?

Horizon - Orçamentos é um sistema para geração de orçamentos, desenvolvido em 2018 por Gabriel Matheus Raimondi, Jaison Joaci Vieira e Leonardo Pokrewecky Salvador, durentes as aulas de programação no Senai de Blumenau. Atualmente, disponível para Java desktop e para web, e com planos para trazer o sistema para mobile em breve!

Qual o objetivo da Horizon - Orçamentos?

Horizon - Orçamentos tem como objetivo facilitar o cadastro de clientes, produtos e fornecedores, e com essas informações cadastradas, possibilitar o usuário a criar um orçamento, para entregar ao cliente ou até mesmo para contabilidade pessoal.
